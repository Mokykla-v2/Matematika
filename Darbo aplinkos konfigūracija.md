### Pavyzdžiai

####Paprasta lygtis

\[
\xi^2+\eta^2+\zeta^2={\rm R}^2
\]

LaTeX kodas šiai lygčiai:

```
\[
\xi^2+\eta^2+\zeta^2={\rm R}^2
\]
```

####Trupmeninė lygtis

\[
\frac{x^2}{(\sqrt{1-v^2})^2}+y^2+z^2={\rm R}^2
\]

LaTeX kodas šiai lygčiai:

```
\[
\frac{x^2}{(\sqrt{1-v^2})^2}+y^2+z^2={\rm R}^2
\]
```
